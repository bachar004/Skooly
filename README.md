# Skooly

Skooly est une application éducative moderne conçue pour faciliter la gestion scolaire et améliorer l’expérience d’apprentissage des élèves & enseignants .

---

## Objectifs de l’application

- Simplifier la gestion des activités scolaires
- Centraliser les informations éducatives
- Améliorer la communication entre élèves, enseignants et administration

---

##  Fonctionnalités principales

### Pour les élèves
- Accès aux cours et supports pédagogiques
- Consultation des emplois du temps
- Suivi des notes et des performances
- Notifications importantes

### Pour les enseignants
- Gestion des cours et des classes
- Publication de supports et devoirs
- Saisie et suivi des notes
- Communication avec les élèves

### admin
- gere tous

## Technologies utilisées

- **Frontend** : Angular bootstrap
- **Backend** : node expressjs
- **Base de données** : bd local mysql
